CSS 第一天

1.CSS 简介

CSS的主要使用场景就是美化网页，布局页面的

1.1 HTML 的局限性

他只关注内容的语义， 丑 ！

1.2 CSS-网页的美容师

CSS 是 层叠样式表 的简称，也是一种标记语言

其最大价值在于让HTML去做结构呈现，而把样式交给CSS，实现结构样式相分离

1.3 CSS 语法规范

规则由两个主要部分组成： 选择器 一条或多条声明

  • 选择器 用于指定CSS样式的HTML标签，花括号内是该对象设置的具体样式
  • 属性和属性值以“键值对”的形式出现
  • 属性是指定的对象设置的样式属性，例如字体大小，文本颜色等
  • 属性和属性值之间用英文":"分开
  • 多个"键值对"之间用英文";"进行区分

1.4 CSS 代码风格

选用 展开格式 进行书写 注意空格保留和大小写

2. CSS 基础选择器

2.1 CSS选择器的作用

说白了，就是用来选择标签的

2.2 选择器分类

选择器 分为 基础选择器 和 复合选择器 两大类

  1. 基础选择器是由 单个选择器 组成的
  2. 基础选择器又包括：标签选择器、类选择器、id选择器、通配符选择器

2.3 标签选择器

是指用 HTML标签名称作为选择器，按标签名称分类，为页面中的某一类标签指定统一的CSS样式

2.4 类选择器

差异化选择不同的标签

语法：

.类名 {

属性1：属性值1

...

}

注意还有多类名的情况

命名要规范！！！！

2.5 id选择器

语法：

#id名 {

属性1：属性值1

...

}

与类选择器的区别在于，id选择器他只供“一个人“使用

2.6 通配符选择器

语法：

*{

属性1：属性值1

...

}

自动给所有元素使用样式

3. CSS字体属性

3.1 字体系列

CSS 使用 font-family属性定义文本的字体系列

p {font-family: "微软雅黑"}

各种字体使用用英文逗号隔开

3.2字体大小

CSS 使用 font-size属性定义字体大小

p {

​ font-size: 20px;

}

px(像素)大小是我们网页最常用的单位

可以给body指定整个页面的字体大小

3.3 字体粗细

CSS 使用 font-weight属性定义粗细

语法：

p {

​ font-weight: 700;

}

常用来说，700=bold粗体，400=normal正常

3.4 文字样式

CSS 使用 font-style属性定义文字样式

语法：

p {

​ font-style: italic;

}

属性值作用
normal 默认值
italic 斜体

3.5 字体复合属性

body {

​ font: font-style font-weight font-size/line-height font-family;

}

说实话，我记不住……

4. CSS 文本属性

4.1 文本颜色

语法：

div {

color: red;

}

三种表示：预定义的颜色值，十六进制，RGB代码

4.2 对齐文本

text-align 属性用于设置元素内文本内容的水平对齐方式

div {

text-align:center;

}

可选择left,right,center三种属性值

4.3 装饰文本

text-decoration 属性用于设置元素内文本的外观

div {

text-decoration:underline;

}

属性值：none(默认), underline, overline, line-through

4.4 文本缩进

text-indent 属性用来指定文本的第一行的缩进

语法：

div {

text-indent:2em;

}

4.5 行间距（行高）

line-height 属性用于设置行间的距离（行高）

5.CSS 的引入方式

5.1 CSS的三种样式表

  1. 行内样式表
  2. 内部样式表
  3. 外部样式表

5.2 内部样式表

<style>

​ div {

color: red;

font-size:12px;

}

</style>

一般将其放在文档的标签中

5.3 行内样式表

语法：

<div style="color: red; font-szie: 12px;">666</div>

5.4 外部样式表

样式单独写到CSS文件中，之后进行引入

使用<link>标签进行引入

语法：

<link rel="stylesheet" href="css文件路径">

CSS 第二天

1. Emmet 语法

1.1 快速生成HTML结构语法

  1. 生成标签直接输入标签名按 tab 键即可
  2. 如果想要生成多个相同标签 加上 * 就可以了
  3. 如果有父子级关系的标签，可以用 >
  4. 如果有兄弟关系的标签，可以用 +
  5. 如果生成带有类名或者id名的，直接写.demo 或者 #two tab键就可以了
  6. 如果生成的div 类名是有顺序的，可以用自增符号 $

1.2 快速生成 CSS 样式语法

CSS 基本采取简写形式即可

1.3 快速格式化代码

VS Code 中：Shift + Alt + F

2.CSS 的复合选择器

2.1 什么是复合选择器

复合选择器是建立在基础选择器之上，对基本选择器进行组合形成的

2.2 后代选择器

语法：

元素1 元素2 { 样式声明 }

其中 元素2 是 元素1 的后代

2.3 子选择器

与后代选择器相比，其只能选亲儿子

语法：

元素1>元素2 {样式声明}

2.4 并集选择器

语法：

元素1，元素2 {样式声明}

2.5伪类选择器

最大特点是用冒号表示，比如：hover

2.6 链接伪类选择器

a: link / visited / hover / active

顺序:LVHA

2.7 focus 伪类选择器

焦点就是光标，一般情况<input>类表单元素才能获取

​ input:focus {

background-color:yellow;

}

3. CSS 的元素显示模式

3.1 什么是元素显示模式

就是元素以什么方式进行显示，一般 HTML 元素氛围 块元素 和 行内元素 两种类型

3.2 块元素

此时该图片与文字底部对齐

注意有时候会出现一个小bug，就是比如说，在建立一个小盒子之后，在这个小盒子里面放上图片，然后图片后加上文字（基线对齐），我们给盒子加上边框。按常理来说，这个div的大小是图片的大小（图片多大它撑得有多大），但是由于后面加了文字，默认基线对齐，这个盒子的边框在下部会有白色空隙

解决方法就是：1.不要是基线对齐即可2.把图片改成 块级元素

7.常见布局技巧

1.margin负值的应用

  1. 让每个盒子margin往左侧移动 -1px 正好压住相邻盒子边框
  2. 鼠标经过某个盒子的时候，提高当前盒子的层级即可（如果没有定位，则加相对定位，如果有定位，则加z-index）

2.文字围绕浮动元素

利用浮动不会压住文字

大盒子里标准流文字，然后加一个左浮动的盒子，文字就在左侧盒子的右侧显示

3.行内块的巧妙运用

父级元素里设置 text-align:center 和 调整 line-height 使其中的行内元素和行内块元素水平垂直居中

8.CSS初始化

不同浏览器对有些标签的默认值是不同的，为了消除不同浏览器对HTML文本呈现的差异，照顾浏览器的兼容，我们需要对CSS初始化

css reset

每个网页都需要CSS初始化

一般的操作有：

  1. 清除内外边距

  2. 斜体不倾斜

  3. 去掉li的小圆点 list-style:none

1.1HTML5 新增的语义化标签

以前布局，我们基本用div来做。div对于搜索引擎来说，是没有语义的

  • ：头部标签
  • :导航标签
  • :内容标签
  • :定义文档某个区域
  • :侧边栏标签
  • ：尾部标签

注意：

  • 这种语义化标准主要是针对 搜索引擎 的
  • 这些新标签页面中可以使用 多次
  • 在 IE9 中，需要把这些元素转换为 块级元素
  • 其实，我们移动端更喜欢使用这些标签
  • HTML5 还增加了很多其他标签
