JQuery

1.概述

1.1 JavaScript 库

封装好的特定的集合（方法和函数）

比如JQuery，方便操作DOM

1.2 jQuery概念

“Write Less, Do More”

用它加快开发速度

链式编程，隐式迭代

2.基本使用

jQuery 的入口函数（两种写法）

  1. 等着DOM结构渲染完毕即可执行内部代码，不必等到所有外部资源加载完成，jQuery 帮我们完成了封装
  2. 相当于原生 JS 中的 DOMContentLoaded
  3. 不同于原生 JS 中的 load 时间是等页面文档，外部的 js 文件、CSS 文件、图片加载完毕才执行内部代码
  4. 更推荐第一种方式

2.1 jQuery 的顶级对象 $

  1. $ 是 jQuery 的别称
  2. $ 同时也是 jQuery 的顶级对象

2.2 jQuery 对象和 DOM对象

  1. DOM 对象：用原生js获取过来的对象就是DOM对象
  1. jQuery 对象：用 jQuery 方式获取过来的对象就是 jQuery 对象

本质：通过$把DOM元素进行了包装（伪数组形式存储）

  1. jQuery 对象只能使用 jQuery 方法， DOM对象则使用原生的 JavaScript 属性和方法

2.3 jQuery 对象和 DOM对象

二者可以相互转换

  1. DOM 转 JQ（两种）
  1. JQ 转 DOM（两种）

3.6 jQuery 样式操作

  1. 参数只写属性名，则是返回属性值
  1. 参数是 属性名，属性值，逗号分隔，是设置一组样式，属性必须加引号，值如果是数字可以不用跟引号和单位
  1. 参数可以是对象形式，方便设置多组样式。属性名和属性值用冒号隔开，属性可以不用加引号。如果是复合属性要用驼峰命名法

类操作与className区别

原生JS 中className 会覆盖元素原先里面的类名。

jQuery 里面类操作只是对指定类进行操作，不影响原先的类名。

  1. 显示语法规范

  2. 显示参数

    • 参数都可以省略，无动画直接显示
    • speed:三种预定速度之一的字符串（“slow”,”normal”,or”fast”）或表示动画时长的毫秒数值（如：1000）
    • easing:(Optional)用来指定切换效果，默认是“swing”,可用参数”linear”
