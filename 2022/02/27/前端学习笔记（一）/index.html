HTML 简介

1. 网页

  1. 什么是网页？

    网站是网页的集合，而网页是网站的“一页”，通常是 HTML 格式的文件，是构成网站的基本元素

  2. 什么是 HTML ？

    是超文本标记语言，用来描述网页

    标记语言是一套标记标签

2. 常用浏览器

​ IE、火狐、谷歌、Safari 和 Opera 等，平时称为五大浏览器

3. Web 标准

​ Web标准是由 W3C（万维网联盟）组织和其他标准化组织指定的一系列标准的集合

3.1 为什么需要Web 标准

​ 浏览器不同，它们显示页面或者排版就有些许差异

3.2 Web 标准的构成

​ 主要包括结构、表现、行为三个方面

​ 主要对应于 HTML、CSS、Javascript

​ 结构类似于身体，表现类似外观装饰，行为类似行为动作,三者中结构最为重要
